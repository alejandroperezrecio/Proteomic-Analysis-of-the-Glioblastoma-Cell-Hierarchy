# Proteomic-Analysis-of-the-Glioblastoma-Cell-Hierarchy
A repository for the code and data used for the Master Thesis project "Proteomic Analysis of the Glioblastoma"

The file "TFM_code.Rmd" is a R Markdown script containing all the code used for data handling, analysis and visualization.

The file "20260413_112142_Glioblastoma y RAD52_Report.xlsx" contains the protein expression data analyzed, which is loaded in the R Markdown script.
